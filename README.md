# gitlang
Repository to list all programming languages github supports.
>The data is from the [trending page](https://github.com/trending)

## Installation
Install the pkg with npm:
```
// v5之前的版本
npm i --save gitlang
```

or yarn

```
yarn add gitlang
```

## Usage
```
import langs from 'gitlang';

console.log(langs);
```
